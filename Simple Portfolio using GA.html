Defining "Fitness": The Sharpe Ratio

We measure if a portfolio is "good" using the Sharpe Ratio. It calculates the return earned per unit of risk taken.

( Portfolio Return - Risk-Free Return ) / Portfolio Risk

A Higher Ratio is Always Better

The Evolutionary Engine

üèÜ

1. Selection

"Survival of the Fittest." The best portfolios (highest Sharpe Ratio) are chosen to become "parents."

üß¨

2. Crossover

"Reproduction." Two parent portfolios are combined to create a new "child" that inherits traits from both.

3. Mutation

"Random Variation." A tiny, random change is made to the child's weights to explore new possibilities.
